# Hierarchical Clustering in Python

## Overview

This project demonstrates the implementation of Hierarchical Clustering using Python. It applies the agglomerative (bottom-up) approach to group data points into clusters based on similarity. The project also visualizes the clustering process using a dendrogram.

## Features

* Implementation of Agglomerative Hierarchical Clustering
* Visualization using dendrogram
* Cluster formation using Ward linkage method
* Graphical representation of clustered data points

## Concepts Covered

* Hierarchical Clustering
* Agglomerative Clustering
* Distance Metrics
* Linkage Criteria (Ward Method)
* Data Visualization

## Technologies Used
